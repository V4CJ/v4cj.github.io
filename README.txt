# 👋 Hi there! I’m Akua Agyare

🎯 **Data Analyst | Business Intelligence Developer | Tech-for-Good Advocate**

Welcome to my **Flutter-powered animated portfolio** — designed with creativity, data-driven storytelling, and developer joy in mind. This site brings together my experience, technical skills, and real-world projects into a sleek, interactive showcase.

---

## 🚀 About the Project

This portfolio is built with ❤️ using **Flutter Web**. It features a fully responsive layout, interactive navigation bar, and animated typewriter text.

---

## ✨ Features

* 🖋️ **Animated typewriter intro**
* 🧭 **Vertical navigation bar** (page links)
* 📊 **Dashboard-style project gallery**
* 🗓️ **Interactive timeline of experience & education**
* 💼 **Dedicated experience & skills pages**
* 📱 **Fully responsive design (desktop/mobile/tablet)**
